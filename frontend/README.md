# 🚀 Zenit Tech - Website Oficial

Repositório do **front-end do site oficial da Zenit Tech**, uma empresa focada em **desenvolvimento de software, consultoria em TI, infraestrutura e soluções tecnológicas sob medida**.

O projeto foi desenvolvido com foco em **alta performance**, **design moderno com Dark Mode nativo** e **animações fluidas**, proporcionando uma experiência moderna e agradável ao usuário.

---

# 🛠️ Tecnologias Utilizadas

Este projeto utiliza ferramentas modernas do ecossistema web para garantir performance, escalabilidade e produtividade no desenvolvimento.

- **React 18+**  
  Biblioteca principal para construção da interface.

- **TypeScript**  
  Tipagem estática que aumenta a segurança e previsibilidade do código.

- **Vite**  
  Bundler extremamente rápido que proporciona um ambiente de desenvolvimento ágil.

- **Tailwind CSS v4**  
  Framework utilitário para estilização rápida, moderna e responsiva.

- **Framer Motion**  
  Biblioteca poderosa para animações, transições e efeitos de scroll.

- **Lucide React**  
  Conjunto de ícones leves, modernos e consistentes.

- **React Router DOM**  
  Gerenciamento de rotas da aplicação (Home, Sobre, Serviços, Contato, Termos e Privacidade).

# ⚙️ Como rodar o projeto localmente

## 📌 Pré-requisitos

Antes de começar, você precisa ter instalado:

- **Node.js**
- **npm** ou **yarn**

Download do Node.js:  
https://nodejs.org/

---

## ▶️ Passo a passo

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/seu-usuario/zenit-tech-site.git
2️⃣ Acesse a pasta do projeto
cd frontend
3️⃣ Instale as dependências
npm install
4️⃣ Inicie o servidor de desenvolvimento
npm run dev
5️⃣ Acesse no navegador

O terminal exibirá um endereço semelhante a:

http://localhost:5173

Abra no navegador para visualizar o projeto.

📁 Estrutura do Projeto

A arquitetura foi organizada para facilitar manutenção e escalabilidade.

src/
│
├── components/      # Componentes reutilizáveis (Navbar, Footer, Cards, etc.)
├── pages/           # Páginas principais (Home, Sobre, Serviços, Contato)
│
├── App.tsx          # Estrutura principal e configuração de rotas
├── main.tsx         # Ponto de entrada da aplicação React
└── index.css        # Estilos globais e importação do Tailwind
📜 Scripts Disponíveis

No diretório do projeto, você pode executar:

Desenvolvimento
npm run dev

Inicia o servidor local de desenvolvimento.

Build de produção
npm run build

Gera a versão otimizada para produção na pasta dist.

Preview do build
npm run preview

Executa um servidor local para visualizar o build de produção.
